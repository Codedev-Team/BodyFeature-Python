 - 사진 경로 지정 (imgPath 변수)  (이미지의 크기는 525x700 픽셀)
 - 콘솔창에 키 입력
 - 콘솔창에 자동으로 구해진 치수 출력 (총장, 어깨너비, 양팔 길이)
 - 마우스 클릭&드래그로 치수 측정 가능
 - 현재 머리, 양쪽 손끝, 발끝, 어깨점 추출 완료
